Aula 1: O que é HTML e para que servem as Tags?

O que foi entendido: Pense no HTML como o "esqueleto" de um site. Ele não é uma linguagem de programação (não tem lógica complexa), mas sim uma linguagem de marcação. Sua função é organizar o conteúdo e dizer ao navegador o que cada parte significa.

Para fazer isso, usamos as Tags. Uma tag é como uma etiqueta que colocamos em um conteúdo. Por exemplo, a tag <p> diz ao navegador: "Isto aqui é um parágrafo". A tag <h1> diz: "Isto é o título mais importante da página". O navegador lê essas etiquetas e monta a página visualmente para o usuário.

Aula 2: A Estrutura Básica de uma Página

O que foi entendido: Toda página HTML segue uma mesma receita de bolo, uma estrutura fundamental. Ela sempre terá estas partes: 

<!DOCTYPE html>
<html>
    <head>
        <!-- Configurações da página (não visíveis) -->
        <title>Título da Aba do Navegador</title>
    </head>
    <body>
        <!-- Todo o conteúdo visível vai aqui -->
    </body>
</html>
  • <!DOCTYPE html>: Avisa o navegador que estamos usando a versão mais recente do HTML.
  • <html>: É a "caixa" que guarda tudo.
  • <head>: A "cabeça" da página. Guarda informações que o usuário não vê diretamente, como o título da aba (<title>), links para CSS, etc.
  • <body>: O "corpo" da página. Tudo que você quer que apareça para o usuário (textos, imagens, links) vai aqui dentro.

Aula 3: Trabalhando com Textos e Parágrafos

O que foi entendido: Para organizar textos, as duas tags mais importantes são as de título (<h1> a <h6>) e a de parágrafo (<p>).

A tag <p> é usada para criar um parágrafo. Cada vez que você usa <p>, o navegador automaticamente cria um bloco de texto com um espaçamento antes e depois, separando-o do resto. Simples assim.

Já os títulos criam uma hierarquia. Pense em um livro: você tem o título do livro (<h1>), os títulos dos capítulos (<h2>) e os subtítulos dentro dos capítulos (<h3>, e assim por diante). Usar essa hierarquia corretamente ajuda tanto na organização visual quanto na otimização para mecanismos de busca como o Google.

Aula 4: Formatando Textos e Dando Ênfase

O que foi entendido: Além de parágrafos e títulos, podemos formatar pequenas partes do texto para dar ênfase ou um significado especial.

  • <strong> e <b>: Deixam o texto em negrito. <strong> indica que o texto tem grande importância.
  • <i> e <em>: Deixam o texto em itálico. <i> é para termos técnicos, e <em> para dar ênfase.
  • <mark>: Funciona como um marca-texto.
  • <small>: Para textos menores, como notas de rodapé.
  • <u>: Para sublinhar um texto.
  • <del>: Para mostrar um texto que foi removido.

Aula 5: Criando Listas

O que foi entendido: Listas são essenciais para organizar informações. Existem dois tipos principais:

1. Listas Não-Ordenadas (<ul>): Usamos quando a ordem dos itens não importa. O navegador coloca marcadores (bolinhas) em cada item. Cada item da lista é definido pela tag <li> (List Item).

2. Listas Ordenadas (<ol>): Usamos para uma sequência de passos, onde a ordem é importante (ex: uma receita). O navegador numera os itens automaticamente (1, 2, 3...). Cada item também é definido com a tag <li>.

Aula 6: Links e Navegação

O que foi entendido: A "mágica" da web está nos links! A tag para criar um link é a <a> (de "âncora").

Ela precisa do atributo href (hypertext reference) para saber para onde o link deve levar. Exemplo: <a href="https://www.google.com">Clique aqui</a>.

Um truque muito útil é o atributo target="_blank". Se você adicioná-lo à tag <a>, o link será aberto em uma nova aba do navegador, o que é ótimo para links externos, pois mantém o usuário na sua página.
